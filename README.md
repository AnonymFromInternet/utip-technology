### Install and Run

#### To run this project You have to have Node.Js

#### [Download Node.Js](https://nodejs.org/en) and install it

#### Then clone this repository via

`git clone https://github.com/AnonymFromInternet/utip-technology.git`

#### And run `npm i`

#### Then `npm start`
